# Invariant-Core - Truth

---

## Definition

Truth is the set of states that survive complete constraint enforcement across all layers of the system.

More formally:

> **Truth is what remains after every invalid possibility has been eliminated, every boundary has been enforced, and all pressure has been resolved without leakage.**

Truth is not asserted, chosen, or interpreted into existence. It is the surviving structure produced by invariant-preserving constraint enforcement.

---

## Core Rule

A state is true if and only if it is:

* invariant-preserving
* globally consistent
* fully admissible across enforced boundaries
* incapable of contradiction, bypass, or invalid re-entry

Truth is therefore not local preference or unresolved interpretation. It is the result of complete survival through the constraint field.

---

## Relation To Pressure

Pressure is unresolved possibility under constraint.
Truth is what remains after pressure resolves without leakage.

Pressure can only contribute to truth when it resolves into invariant-preserving form. If pressure leaks, truth has not closed. If pressure is eliminated, the invalid possibility no longer participates in truth.

---

## Relation To Closure

Closure is the absence of invalid survival paths.
Truth is the set of states that remain inside that closure.

A system has not reached truth while invalidity can still:

* persist
* propagate
* bypass a seal
* re-enter through recursion
* survive as ambiguity
* remain hidden as latent inconsistency

---

## Layer Behavior

Truth must hold across the full invariant stack.

A state cannot be true if it satisfies a higher layer while violating a lower one. Lower-layer contradiction, impossibility, incoherence, or invalidity defeats higher-layer acceptance.

As layers descend, truth becomes less negotiable because constraint and authority converge.

---

## Operational Test

A state can be treated as true only when all of the following hold:

* all applicable constraints are satisfied
* all relevant boundaries have enforced admissibility
* no invalid state can survive or re-enter
* no unresolved pressure remains hidden or leaked
* no lower-layer invariant is violated
* the state remains consistent under valid transformation

---

## Final Compression

> **Truth = invariant-preserving survival after complete constraint enforcement.**

> **Truth is what remains when invalid possibility has nowhere left to survive.**
