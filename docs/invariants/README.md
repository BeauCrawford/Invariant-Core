## Role

Invariant documents specify the rules that foundation concepts must satisfy:

* how downstream invariants remain contained by the upstream shell
* what valid structure requires
* where constraints become authoritative
* how invalidity is prevented from surviving
* how pressure resolves into closure
* how lower-layer invariants bind higher-layer systems
* how truth survives complete enforcement
